Isle Of Wight Gin

This was a project in which I had the oppurtunity to create a bus stop advert for IOW Gin

  • ClientIsle Of Wight Gin
  • RoleDesigner
  • Year2026
  • ToolsPhotoshop, Illustrator

In May 2026, Isle Of Wight Gin asked my college if they would like to complete some live client work for them, in which they wanted bus stop adverts to promote their product

When I was told about the project I started experimenting with ideas and also using any previous work to help me, in doing this I found a Football Graphic I made that used a cloud background. This previous piece was my main base to help complete both posters as I found the cloud background helped the bottles and text stand out.

I found this project really helpful as it allowed me to see what working for a real client was like and how the design world worked
